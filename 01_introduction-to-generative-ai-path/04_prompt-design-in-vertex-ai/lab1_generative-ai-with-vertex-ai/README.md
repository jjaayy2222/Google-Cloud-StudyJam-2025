## 학습 내용
1. `Be concise` - 간결한 프롬프트 작성
2. `Be specific` - 구체적이고 명확한 요청
3. `Ask one task at a time` - 한 번에 하나씩
4. `System instructions` - 가드레일 설정
5. `Zero-shot`, `One-shot`, `Few-shot`
